# 運動量と力積シミュレーター

Ft = mv₂ − mv₁ を体感するインタラクティブ物理シミュレーター

🔗 **公開URL**: https://momentum.meetupsensei.com/

## 概要

物体に力 *F* を時間 *t* だけ加えたときの「力積（Ft）」が運動量の変化（mv₂ − mv₁）と等しくなる様子を、アニメーションとグラフで体感できる物理シミュレーターです。質量・力・時間・速度のいずれかを自動計算するモードを切り替えながら、衝突・跳ね返り・エアバッグなどの実例プリセットで運動量保存の法則を直感的に理解できます。ダークモード・ライトモード、日本語/英語の表示切り替えにも対応しています。

## 使い方

- 上部の「デフォルト」「跳ね返り」「野球（バット）」「エアバッグ」「キャッチ」プリセットから状況を選択する
- 質量 *m*、力 *F*、時間 *t* などのパラメータをスライダーや入力欄で調整する
- モードタブで「v₂を自動計算」「Δpを自動計算」「v₁を自動計算」を切り替える
- アニメーションとグラフで運動量の変化と力積の関係を観察する
- 右上のボタンでライト/ダークモードや日本語/英語表示を切り替える
